# Michelin-Pull
The main purpose of this is to generate something that can be downloaded/printed because they won't do it.

# Instruction
Paste the js code into the console of the Michelin Guide website and press enter. Some things you may want/need to change:
- Line 1, the url contain the api key the website uses to pull their data. This may update and you'll need to find the new one using the console
- Line 14 and line 36 determines the language of what you pull. (E.g. change the last 2 letters "en" to "fr" for French)
- Line 15 and line 37 determines the location in latitude, longitude
- Line 17 and line 39 determines the radius
- Line 54 determines the maximum number to get. By default it is everything

Note that if I listed 2 lines for something, you need to change both otherwise it may behave unpredictably.
